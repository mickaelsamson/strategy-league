# Moonveil Ascend

Moonveil Ascend is a dark fantasy RTS prototype built in Unity.

## Vision

A premium isometric/top-down RTS for Strategy League.

Core gameplay loop:

Gather resources → Build base → Train units → Attack enemy base → Win.

## Visual Direction

Gameplay readability reference:
- clear isometric RTS camera
- central base layout
- visible workers harvesting nearby resources
- clear resource nodes
- top resource bar
- objective panel
- minimap
- selected unit/building panel
- command/build menu

Moonveil identity:
- dark fantasy tone
- black, gold, deep purple, blue crystal accents
- mystical buildings and units
- glowing Essence crystals
- premium ornate UI

## Phase 1A Goal

Create the first playable RTS foundation:

- RTS camera
- player main base
- enemy main base
- 3 workers
- Essence, Stone, Nature resource nodes
- basic selection
- right-click movement
- worker gathering loop
- top resource bar
- simple UI placeholders

No multiplayer yet.
No advanced combat yet.
No final art yet.
